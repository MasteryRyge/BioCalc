
Este projeto é um trabalho escolar que se consiste em uma plataforma web que implementa todas as funcionalidades de uma ferramenta chamada BioCalc - Calculadora de créditos de descarbonização para biocombustíveis sólidos. A ferramenta original pode ser encontrada no seguinte link:

https://www.grupoengs.com.br/produtos

O projeto foi realizado utilizando React, Node, Bootstrap e MongoDB
